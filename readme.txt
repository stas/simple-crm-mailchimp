Adds MailChimp integration to Simple CRM

== Description ==

Simple CRM is a WordPress plugin that lets you define custom fields to extend user profiles and it is also a framework for integration with all kind of CRM API webservices. MailChimp addon lets you sync the defined profile fields with a list using it's API.

Plugin sponsored by [Vivanista.com](http://vivanista.com)

== Installation ==
